# 📊 Sales Analysis Project

## 📁 Overview
Exploratory Data Analysis (EDA) on a global orders dataset covering **July 2012 to May 2014**,
analyzing sales patterns across cities, countries, customers, and time.

## 📂 Structure
- `Bivariate_analysis.ipynb` — Main analysis notebook

## 🔍 Analysis Covered
- **Univariate** — Distribution of total amount, cities, countries
- **Bivariate** — Country vs revenue, city vs revenue, customer analysis
- **Multivariate** — Monthly trends, country performance over time

## 💡 Key Findings
- 🇩🇪 Germany & 🇺🇸 USA lead in both order count and revenue
- 🇦🇹 Austria has only **2 customers** — explaining its volatile sales pattern
- **London** dominates city-level orders
- Revenue shows a **clear upward trend** from 2012 to 2014
- Last month (May 2014) is **incomplete** — only 6 days of data

## 🛠️ Tools Used
- Python, Pandas, Plotly, Seaborn, Matplotlib
